SpringSide is a spring based JavaEE application reference architecture. 

It has 3 major examples:

1. mini-web focus on the minimal web application about domain object CRUD.
2. mini-service focus on the effcetive web service in SOAP and REST style.
3. showcase demonstrate many many technology used in a JavaEE application.
 
-------------------------------
Please see the wiki (http://github.com/springside/springside4/wiki) for the details.
